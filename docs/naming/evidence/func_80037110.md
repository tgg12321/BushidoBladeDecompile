## Why this name

* `special_cam_*` reflects the SpecialCam global access.
* `lookup_table_entry` reflects the (a) indexed lookup of a per-row
  descriptor, (b) the conditional scheduling action it performs.
* Address suffix preserved because the exact role of `D_8008F13C`
  table entries is still pending decomp -- a clearer name like
  `special_cam_schedule_deferred_read` would be better once the
  D_8008F13C structure is understood.
