# func_80085064 -- naming evidence

- Address: `0x80085064`
- Source: `asm/funcs/func_80085064.s`
- Proposed name: `seq_channel_byte_advance_80085064`
- Confidence: **medium**

## Body

Indexes the same `D_80106F28[bank][channel*0x58]` block. Reads the next byte from
ch[0x00] (advancing the pointer), then if MSB set walks a 7-bit varint chain
(repeatedly reads next byte, OR-accumulates the low 7 bits into the value) until
MSB clear. Finally writes the resulting value*2 into ch[0x88]. Returns 0 if the
very first byte was 0.

## Evidence

Classic 7-bit-varint reader: while MSB set, accumulate low 7 bits, shift left 7, OR in next byte. Output written to ch[0x88] which is presumably the next delta-time/duration register.

## Why this name

SEQ delta-time decoder. Reads a variable-length quantity from the channel byte stream and stashes the (sample-count-doubled) result into ch[0x88].
