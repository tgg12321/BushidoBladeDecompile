## Why this name

Voice subsystem state code. Tested against {0, 3, 5} in the voice
controller and the per-character voice-data fetcher. Companion to
`g_game_mode_code` (D_800A3834) -- one is the global game state, the other
is the voice subsystem's sub-state.
