# D_800A3858 -- naming evidence

- Address: `0x800A3858`
- Proposed name: `g_round_frame_counter`
- Confidence: **high**

## Storage / access pattern

.bss word.  Counts up by 0x384 (900 = 30 seconds at 30Hz) per round;
also converted to centiseconds for HUD display: `(val % 30) * 100 / 30`.

## Why this name

Round/match elapsed-frame counter.  The `% 30 * 100 / 30` conversion
extracts centiseconds from the 30Hz frame count for HUD display.
The 0x384 (900 frames = 30 seconds) increment is the per-round
time-bonus or similar batched bump.
