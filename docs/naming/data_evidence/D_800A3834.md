extensively in mode-machine checks.

## Why this name

Active game mode / state code. The handful of values (0/1/9/0xD/0x1B) and
the assignment pattern in scene_teardown variants identify this as the
primary game-mode enum -- equivalent to a state-machine cursor for the
main game loop.
