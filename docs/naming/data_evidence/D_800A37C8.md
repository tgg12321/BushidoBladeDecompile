## Why this name

Sub-state of the analog-pad handler.  Coupled with g_pad_analog_mode
(D_800A31F4) -- one selects high-level mode, the other selects sub-state
within a mode.  Value 0 forces analog-mode 5, value 3 takes the
set_state_7 branch.
