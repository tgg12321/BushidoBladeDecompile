# D_800A36F6 -- naming evidence

- Address: `0x800A36F6`
- Proposed name: `g_active_character_id`
- Confidence: **medium**

## Storage / access pattern

.bss byte / halfword.  Used to identify the active character; passed
to gnd_init_80041688 and compared to setters' arg0.

## Usage in src/

src/code6cac.c -- `func_8001B690`:
  if (D_800A36F6 != arg0) ...

src/code6cac.c -- `func_8001E404`:

## Why this name

Currently-active character id (selected character / opponent), passed
to per-character ground init.  The 'change detection'
`if (D_800A36F6 != arg0)` ensures expensive reinit only on character
swap.
