- Address: `0x800F19B8`
- Proposed name: `g_vsync_timeout_deadline`

## Storage / access pattern

.bss word.  Set to `sys_VSync(-1) + 0x3C0` (current VSync count + 960
frames = 16 seconds at 60Hz or 32 seconds at 30Hz); compared to
current VSync to detect timeout.

## Why this name

VSync-frame deadline for a timeout (16-32 seconds wall time).  Used
as the bailout condition in a busy-wait/poll loop.
