## Storage / access pattern

Array of pointer entries, indexed by `((a0 << 16) >> 14)` (i.e. a 16-bit
value scaled by 4 for word-stride).  Each entry points to a per-channel
control block of stride 0xB0 (176 bytes).

## Why this name

Base table of SEQ playback banks.  Outer index = bank id (s16 from a0),
inner stride 0xB0 is the per-channel control block size visible in the
seq_channel_state_init/clear/byte_advance helpers (func_8008{5114,5210,5064}).  The bank table is touched by `spu_SetMotionState` (initialiser) and
addressed by the per-bank gauge/state lookups.
