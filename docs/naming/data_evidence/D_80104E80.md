# D_80104E80 -- naming evidence

- Address: `0x80104E80`
- Proposed name: `g_blood_spot_timer`
- Confidence: **medium**

## Storage / access pattern

Single .word in .bss, set to small frame counts (0x3C == 60, 120 == 2s),
decremented via the timer/alarm handler.

## Usage in src/

src/main.c -- `SetBloodSpot`:
  D_80104E80 = t;
  D_80104E80 = 120;

src/ings2.c -- `func_80083A48`:
  D_80104E80 = 0x3C;

src/ings2.c -- `sys_Shutdown`: extern.

## Why this name

Visible-blood-spot countdown timer.  `SetBloodSpot` arms it from a passed
value or a hardcoded 120 frames; the alarm-side helper resets it to 60.
Counts down to expire the on-screen blood splatter.
