# D_800A2CD4 -- naming evidence

- Address: `0x800A2CD4`
- Proposed name: `g_spu_pending_irq_mask`
- Confidence: **high**

## Storage / access pattern

.bss word.  SPU IRQ mask; tested as `D_800A2CD4 & 1` for pending.
Reset by spu_InitEx.

## Usage in src/

src/main.c -- `spu_WriteReg`: extern s32 D_800A2CD4;

src/main.c -- `spu_InitEx`:
  D_800A2CD4 = 0;

src/main.c -- `coli_HitPauseKatana_2`:
  if (D_800A2CD4 & 1) ...

## Why this name

SPU pending-IRQ / pending-operation mask.  Different from
g_spu_voice_mask (which holds key-on/off bits); this one holds
request/done flags for SPU IRQ dispatch.
