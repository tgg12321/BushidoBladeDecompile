# D_800F19BC -- naming evidence

- Address: `0x800F19BC`
- Proposed name: `g_irq_dispatch_counter`
- Confidence: **medium**

## Storage / access pattern

.bss word in system.c.  Reset to 0 and post-incremented by
cpu_side_move_dir_4 (which is misnamed -- it's actually an IRQ
dispatcher path per the surrounding code).

## Usage in src/

  D_800F19BC = 0;
  cnt = D_800F19BC;

## Why this name

Sequence counter for IRQ dispatch tracking.  The `- -1` is GCC's idiom
for `cnt + 1` (avoids strength-reduction).  Lives in system.c which is
the irq/cdrom/spu/sys helper file.
