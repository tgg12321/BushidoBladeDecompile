## Why this name

Pointer to the currently-active stage data blob. The collision-center
fields at +0x4C/+0x54 are read by `stage_InitCollision`; replay-camera
code reads +0x10. This is the standard 'current stage descriptor' pointer
loaded from the stage data file.
