# D_800A37DC -- naming evidence

- Address: `0x800A37DC`
- Proposed name: `g_memcard_event_io_complete`
- Confidence: **high**

## Storage / access pattern

.bss word holding a BIOS event handle for memcard I/O completion
(class 0xF4000001, mode 4).

## Usage in src/

src/code6cac_c.c -- `gnd_close_8004939C`: extern s32 D_800A37DC;

src/code6cac_c.c -- `pad_press_control` (which is actually memcard_event_init):
  D_800A37DC = bios_OpenEvent(0xF4000001, 4, 0x2000, 0);
  bios_EnableEvent(D_800A37DC);

## Why this name

Memcard I/O-complete event handle.  Class 0xF4000001 = memcard,
mode 4 = I/O completion notification.  One of the 8 events opened
by memcard_event_init (formerly mis-named `pad_press_control`).
