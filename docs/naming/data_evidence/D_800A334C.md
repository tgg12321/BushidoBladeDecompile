# D_800A334C -- naming evidence

- Address: `0x800A334C`
- Proposed name: `g_motion_play_countdown`
- Confidence: **high**

## Storage / access pattern

Single u8 in .bss.  Set to 0x5A (90 frames == 1.5s at 60Hz), decremented
each tick, gated on reaching zero or on a voice-state bit set.

## Usage in src/

src/code6cac_c_mid.c -- `motion_SetMotion`:
  D_800A334C = 0x5A;
  D_800A334C--;
  if (((u8)D_800A334C) == 0 || (D_80102794 & 0x100010)) ...

## Why this name

Frame-counter watchdog for motion-state-machine timeout.  The 0x5A value
(90) is a one-second-and-a-half timeout; the OR with `g_seq_active_voice_mask`
(D_80102794) means a state change is forced either by timeout or by a
specific voice bit going active.
