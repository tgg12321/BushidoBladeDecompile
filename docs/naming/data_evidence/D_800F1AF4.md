# D_800F1AF4 -- naming evidence

- Address: `0x800F1AF4`
- Proposed name: `g_packet_data_dma_addr`
- Confidence: **high**

## Storage / access pattern

Volatile .bss word; address-aliased pointer used by SetPacketData
(which is the DMA-list submission function for GPU/SPU packets).

## Usage in src/

src/main.c -- `cpu_side_move_dir_3`: extern s32 D_800F1AF4;

src/main.c -- `func_8008C184`:
  D_800F1AF4 = a1;

## Why this name

Volatile cell holding the current DMA-list address for packet-data
transfers.  Updated by func_8008C184 before SetPacketData kicks the
DMA.
