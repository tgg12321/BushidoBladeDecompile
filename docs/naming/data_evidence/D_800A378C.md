## Why this name

Base pointer for an array of palette/colour entries indexed by `idx`.
The 4-byte stride matches a CLUT-entry table (one s32 per palette).
Bound at runtime (it's a pointer field, not a static array), suggesting
the palette table is loaded from disc into a malloc'd buffer.
