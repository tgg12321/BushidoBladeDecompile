# === PSX BIOS trampolines (canonical hand-coded asm) ===
# Pattern (3 instructions):
#     addiu $t2, $zero, 0xA0/0xB0/0xC0   ; load BIOS vector
#     jr    $t2                          ; tail-call to kernel
#     addiu $t1, $zero, FUNC#            ; delay slot: function number
# These cannot be expressed in C. The PSX BIOS calling convention requires:
#   (1) a tail-call jr (not jalr) to a fixed kernel vector address,
#   (2) the BIOS function number pre-loaded into $t1 specifically,
#   (3) the $t1 load placed in the jr's branch-delay slot.
# GCC 2.7.2 emits standard jal with argument-passing convention; it has
# no way to produce these 3 bytes. Authorized en masse 2026-05-15 after
# manual review of all 39 candidates against asm/funcs/<name>.s.
